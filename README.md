
# Pathfinder Visualizer

A tool to visualize the A* pathfinding algorithm in a 20x20 grid. The user can interact with the grid, setting start and end points as well as obstacles, to see the algorithm in action.

## 🛠 Tech Stack
- Python
- Tkinter (for GUI)

## 📌 Features
- ✔️ Visualizes the A* pathfinding algorithm in a 20x20 grid.
- ✔️ Interactive grid with customizable start/end points and obstacles.
- ✔️ Real-time visualization of the algorithm’s pathfinding process.
